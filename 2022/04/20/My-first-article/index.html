hexo博客搭建-.-

内容：
1，安装NodeJs, 有node和npm
2，安装Git，将git../bin配置到环境变量path和系统变量path，配置好用户名（与github的ID一致）和邮箱
$ git config –global user.name “Your Name”
$ git config –global user.email “email@example.com“
3，安装hexo，npm install -g hexo-cli
4，hexo init–>初始化到blog目录中 ，
hexo s–>开启
hexo n “name”–>创建文章
souce/_posts/“name”–>修改
hexo clean–>刷新
hexo g–>生成本地地址http://localhost:4000
5，修改config.yml，
deploy:
type: git
repo: https://github.com/SenCheng98/SenCheng98.github.io.git（new repository）
branch: master
6，hexo d–>配置hexo，部署到internet中
7，https://SenCheng98.github.io.git-->访问博客
